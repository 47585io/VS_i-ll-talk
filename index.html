Blob对象

HTML5的 Blob 对象用于存储二进制数据，还可以设置存储数据的MIME类型，其他HTML5二进制对象继承Blob对象。

【实例3】关键代码 

function getSize(){
	for (let file of document.getElementById("file").files)
		console.log(file.size)
		console.log(file.type)
		//因为file对象继承自Blob，所以也拥有size和type属性
}
